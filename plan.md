# 项目计划书

## 目的

修改现有项目，实现一个宣传商品的网站，用以宣传和售卖商品

## 单一商品

塘栖枇杷

## 技术架构

使用React + TS + Vite + Tailwind CSS + PostCSS（Radix UI + shadcn）

## 运行环境

本机运行，配置为：macOS 26.2 + m1max + 32G 内存

## 风格

读取“网站风格.md”内容，主要是apple风

## 图片资源

/static下所有图片，比例和形状按需切割，按需可以生成图片。

## 要求

样式兼容PC和H5

## 内容

> 以下内容，只有第一屏是严格要求是可视范围的1屏高，其他内容可以超过1屏高

### 第一屏

首屏是一颗黑白的枇杷，在一块白色带灰的背景上，枇杷自身轻微摆动，要有呼吸感。然后鼠标移动，枇杷会跟随鼠标位置“左右滚动”，要凸显动画效果。可以使用canvas制作，也可以用其它第三方框架，你自选。

### 第二屏及一下

图文结合，介绍枇杷的功效，营养，口味等价格。

目前价格分如下三类：

1、上门自提12/斤

2、下地采摘和购买15/斤

3、顺丰邮寄，20/斤，不包邮。邮费和快递单号可提供截图

最后附带定位，可显示地图：GCJ-02坐标系，120.179454,30.493722
